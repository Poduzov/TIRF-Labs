# TIRF-Labs
Laser controls products for microscopy applications

Arduino IO Shield
- 2 Digital hardware interrupt inputs
- 2 Digital outputs
- 8 Analog Outputs, 2xAD5625R 12bit DAC, 0..5V
- Arduino pins A0-A5 wired to an auxiliary conector
